### Installation
```
git clone https://github.com/VasiliyKochurko/ostis-cybersport.git
cd ostis-cybersport/backend
pip install -r requirements.txt
python main.py
```

#### Open new terminal 
```
cd frontend 
npm i 
npm start
```
#### Open http://localhost:3000/
